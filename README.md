# 谭颖恒 H5 简历

这是最终可上传版本，面向 GitHub / Vercel 静态部署。

## 文件说明

- index.html：网页入口，直接部署该文件即可访问。
- assets/：页面图片资源，必须和 index.html 一起上传。
- upload.bat：自动提交并推送到 GitHub 的脚本。
- .gitignore：Git 忽略规则。
